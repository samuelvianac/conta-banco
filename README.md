# Desafio Conta Banco

Projeto desenvolvido em Java para praticar sintaxe e entrada de dados.

## Funcionalidades
- Entrada de dados via terminal
- Criação de conta bancária simples
- Exibição de informações do cliente

## Tecnologias
- Java

## Execução
O programa solicita:
- Número da conta
- Agência
- Nome do cliente
- Saldo

E exibe a mensagem final com os dados informados.
